# hiring_test_project
JobKred hiring test project

1. Run command `composer install`
2. Set up env file and add information from `.env.example` file
3. Run migration
```
$ php artisan migrate
$ php artisan db:seed
```
4. Install gulp
```bash
$ npm install gulp-cli -g
$ npm install gulp -D
```
5. Install node module 
```
npm install
```
